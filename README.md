# EpicFight GodSkill Mod

EpicFight GodSkill Modによって追加される全スキルの詳細な機能表です。ゲーム内のツールチップには記載されていない強力な「隠し仕様」も含めて詳細に記載しています。

| スキル名 | スロット | ステータス補正・属性値変化 | 詳細な機能・効果・隠し仕様 |
| :--- | :--- | :--- | :--- |
| **ゴッドスキル**<br>(Godskill) | パッシブ | ・最大スタミナ +20<br>・スタミナ回復量 +20<br>・最大体力 +20 (ハート10個分)<br>・攻撃力 +50%<br>・攻撃速度 +50%<br>・オフハンド攻撃速度 +50% | **[全能パッシブ]**<br>・**消費スタミナゼロ**: あらゆるスキルのスタミナ消費が完全になくなります。<br>・**武器固有スキル常時使用可**: 武器固有スキルのスタック(回数)が使用後即座に最大まで回復します。<br>・**モーションキャンセル**: 攻撃モーション中であっても、ドッジ(回避)やガードでいつでも割り込んでキャンセル可能になります。<br>・**無限スタンシールド**: ダメージを受けると即座にスタンシールドが最大値(1000)まで回復し、実質的に攻撃を受けても怯まなくなります。<br>・**継続回復**: 2秒ごとに体力が6回復します。 |
| **ゴッドリーチ**<br>(God Reach) | パッシブ | ・攻撃力 +50% | **[超長距離攻撃]**<br>・**リーチ5倍**: 攻撃の届く距離（突進リーチ含む）が通常の5倍に拡張されます。<br>・**広範囲な当たり判定 (隠し仕様)**: 通常の武器の当たり判定に加え、**自身の周囲15ブロック以内**にいる敵対可能な全エンティティに強制的にダメージを与えます。<br>・**継続回復**: 2秒ごとに体力が3回復します。 |
| **エンジェルスキル**<br>(Angel Skill) | パッシブ | ・攻撃力 +50% | **[天使の加護]**<br>・**継続回復**: 2秒ごとに体力が3回復します。 |
| **使徒スキル**<br>(Apostle Skill) | パッシブ | ・攻撃力 +50% | **[使徒の力]**<br>・**継続回復**: 2秒ごとに体力が2回復します。 |
| **祝福**<br>(Blessing) | アイデンティティ<br>(Revelation) | なし | **[神の祝福]**<br>・**継続回復**: 2秒ごとに体力が5回復します。<br>※Revelation系のスロットに装備するスキルです。 |
| **飛行**<br>(Flying) | ムーバー<br>(Mover) | なし | **[クリエイティブ飛行]**<br>・サバイバルモードでもクリエイティブモードのように空を飛ぶことが可能になります。<br>・**継続回復**: 2秒ごとに体力が3回復します。 |
| **神回避**<br>(God Dodge) | ドッジ<br>(Dodge) | なし | **[強化版ロール回避]**<br>・**超高速移動**: 回避時の移動速度が通常の5倍になります。<br>・**消費スタミナゼロ**: スタミナを一切消費せずに回避し放題になります。<br>・**継続回復**: 2秒ごとに体力が3回復します。<br>※前転および後転（ロールモーション）の回避アクションです。 |
| **神移動**<br>(God Step) | ドッジ<br>(Dodge) | なし | **[強化版ステップ回避]**<br>・**超高速移動**: 前後左右4方向へのステップ時の移動速度が通常の5倍になります。<br>・**消費スタミナゼロ**: スタミナを一切消費せずにステップし放題になります。<br>・**継続回復**: 2秒ごとに体力が3回復します。<br>※全方向対応のステップモーションの回避アクションです。 |
| **パーフェクトガード**<br>(Perfect Guard) | ガード<br>(Guard) | なし | **[完全無欠の防御]**<br>・**全方位防御**: 前方からの攻撃だけでなく、**360度全方向からのあらゆる攻撃**をガードできます。<br>・**防御不可の貫通攻撃もガード (隠し仕様)**: 矢などの飛び道具はもちろん、本来なら防御できない貫通・ガード不可設定の攻撃すらもガードします。<br>・**ガードブレイク無効 (隠し仕様)**: ペナルティが0に設定されており、ガードによってスタミナが減らず、絶対にガードブレイクされません。<br>・**環境ダメージ無効 (隠し仕様)**: ガード中は、溶岩、落下ダメージ、奈落など、本来EpicFightが干渉しない**環境からのダメージも完全に無効化**します（ガードのパリィ演出・音が発生します）。<br>・**回復量3倍 (隠し仕様)**: ガード中、自然回復やポーションなどを含む**自分への全ての回復効果が3倍**になります。<br>・**満腹度回復 (隠し仕様)**: ガードを維持している間、**1秒ごとに満腹度と隠し満腹度(サチュレーション)が6回復**します。<br>・**継続回復**: 2秒ごとに体力が3回復します。 |
